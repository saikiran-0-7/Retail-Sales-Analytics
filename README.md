# Retail Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data to uncover insights into sales performance, customer behavior, profitability, and regional trends.

The project demonstrates an end-to-end analytics workflow using SQL, Python, MySQL, and Power BI.

---

## Business Problem

Retail businesses generate large amounts of transactional data. The objective of this project is to analyze historical sales data to identify:

- Top-performing regions
- Best-selling product categories
- Most valuable customers
- Sales trends
- Profitability
- Customer segments

---
